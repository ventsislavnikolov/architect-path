# Architect Path — Principal Dev → (AI) Software Architect

A personal, bilingual (EN/BG) learning workspace for the transition from Principal Software
Developer to **(AI) Software Architect**, built as a structured course of short, beautiful lessons.

## Mission
Earn an internal promotion to (AI) Software Architect — broader influence, the title, riding the
AI shift, and more design / less hands-on code. Full context in [`MISSION.md`](./MISSION.md).

## How it's organized
| Path | What it is |
|------|------------|
| [`MISSION.md`](./MISSION.md) | Why this course exists — grounds every lesson |
| [`RESOURCES.md`](./RESOURCES.md) | High-trust sources (books, articles) behind the lessons |
| [`NOTES.md`](./NOTES.md) | Teaching preferences & open questions |
| `lessons/` | Self-contained HTML lessons (`*.html` = EN, `*.bg.html` = BG) |
| `reference/` | Quick-reference docs — glossary, templates |
| `learning-records/` | Decision records capturing what was learned & why |
| `assets/` | Shared stylesheet + reusable widgets (quiz, etc.) |
